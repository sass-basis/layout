# sass-basis-layout
This is a css module for sass-basis.

## Basis
* Repository: https://github.com/sass-basis/basis/
* Documents : http://sass-basis.github.io/basis/

## Get Started
1. `$ npm install sass-basis`
2. Import `src/scss/layout/_container.scss` your Sass/SCSS.
3. Import `src/js/fixed-header.js` when using `.l-container--overlay-header`

## Browser support
Modern Browser and IE10+
